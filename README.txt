== Description ==
Make custom lists with Images, Font-awesome icons, and Texts with Custom Color, Font-size style. that will display in the sidebar.


== Installation ==

1. Download the plugin, extract the zip file.
1. Upload the custom-sidebars folder to your /wp-content/plugins/ directory.
1. Active the plugin in the plugin menu panel in your administration area.
